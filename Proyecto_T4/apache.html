Apache

Apache es un poderoso servidor web, es completamente libre, ya que es un software Open Source y con licencia GPL. Una de las ventajas más grandes de Apache, es que es un servidor web multiplataforma, es decir, puede trabajar con diferentes sistemas operativos y mantener su rendimiento.

Principales características de Apache

  • Soporte de seguridad SSL y TLS.
  • Puede realizar autentificación de datos utilizando SGDB.
  • Puede dar soporte a diferentes lenguajes, como Perl, PHP, Python y tcl.

Instalación y configuración de Apache

  1. Abrimos el servidor y ejecutamos el comando apt-get install apache2
  2. Ejecutamos el comando service apache2 start para iniciar Apache
  3. Si queremos comprobar el estado de Apache, ejecutamos el comando service apache2 status
  4. Ejecutamos el comando nano index.html y procedemos a crear la página html
  5. Damos a ctrl+o para guardar y ctrl+x para salir
  6. El servidor apache está listo y configurado

REQUISITOS RECOMENDADOS

  • Linux kernel version 2.4 o posterior, glibc2 version 2.3.2 o posterior (inicio con OOo 3.3 glibc2 version 2.5 o mayor) 256 Mbytes RAM (512 MB recomendado)
  • Espacio de disco de 400 Mbytes
  • X-Server con 1024 x 768 o mayor resolucion con 256 colores
