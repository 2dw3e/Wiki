Ubuntu Server

Ubuntu Server es una variante de Ubuntu que sale con cada versión y está dedicada especialmente para su uso en servidores. El uso de Ubuntu como servidor se ha extendido mucho en los últimos años, sobretodo para el uso de servidores web, de un modo tanto particular como profesional.

Creación de un container con Ubuntu Server

  1. Accedemos a proxmox mediante la web (https://10.9.53.113:8006/).
  2. Accedemos con nuestro usuario (root / admin)
  3. En almacenamiento local, le damos a ‘Contenido’ y a ‘Plantillas’ *
    4. * Si la plantilla que buscamos no está procederemos a instalarla:
    3.1.- Le damos a click derecho en nuestro servidor ‘2dw3e’ y le damos a ‘<_Shell’
    3.2.- Se abrirá la consola del servidor, y ejecutaremos el siguiente comandos: pveam update

  4. Volvemos a Proxmox y realizamos el paso 3 de nuevo.
  5. Seleccionamos la plantilla que queramos y le damos a ‘Descargar’. En nuestro caso hemos instalado ubuntu-17.04-standard_17.04-1_amd64.tar.gz
  6. Una vez descargado, en ‘Contenido’ aparecerá nuestra plantilla. Le damos a ‘Crear CT’
  7. Rellenamos con los datos que haga falta.
  8. Para ver la versión que hemos instalado, le daremos doble click sobre el nombre del container
  9. Se abrirá la consola y ejecutaremos el comando lsb_release -a
  10. Ahora ejecutamos apt-get update y apt-get upgrade
  11. Si queremos realizar por ejemplo un ifconfig, tenemos que instalar net-tools con el comando apt install net-tools

REQUISITOS RECOMENDADOS

  • Procesador Procesador x86 a 1 GHz
  • Memoria RAM 1 GB
  • Tarjeta gráfica Super VGA (800 x 600)
  • Espacio en disco Disco Duro de 15 GB
  • Unidades Una de las siguientes:Lector de CD-ROM, puerto USB o tarjeta de red.
  • Otros dispositivos Monitor Super VGA ( resolución 800 x 600 o más), teclado y ratón.La conexión a Internet no es necesaria, pero puede ser útil.
