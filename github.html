GitHub

Github es una plataforma creada para facilitar el desarrollo colaborativo de software, permite alojar proyectos en la web gratuitamente, por lo general de forma pública, aunque se puede alojar los proyectos de modo privado, pagando una pequeña suscripción mensual.

Instalación y configuración de Git

  1. Instalamos Git. Para ello ejecutamos el comando

    apt-get install git

  2. Ahora configuraremos Git:
    El primer paso es iniciar Git. Nos situamos en la carpeta donde queramos alojar el proyecto (en nuestro caso /var/www/html) y ejecutamos el comando

    git init

  3. Ahora le asignamos el repositorio que hemos creado previamente en la página de GitHub. Ejecutamos el comando

    git remote add origin https://github.com/2dw3e/Wiki.git

  4. Ahora nos identificamos. Ejecutamos el comando

    git config --global user.email 2dw3euskera@gmail.com

  5. Para subir todos los archivos que hay en la carpeta ejecutamos

    git add.

  6. Para crear un commit ejecutamos

    git commit -m “nombre”

  7. Para subir los cambios realizados a GitHub ejecutamos

    git push -u origin master

  8. Introducimos el usuario y la contraseña de GitHub
  9. Una vez que hayamos hecho algún cambio y lo hayamos subido a GitHubt, si queremos descargar el proyecto completo, en el servidor, ejecutamos

    git pull origin master

OBJETIVOS GITHUB

  • Saber dónde obtener información sobre git.
  • Conocer el estado e historial de nuestros proyectos.
  • Modificar y recuperar estados anteriores.
  • Solucionar conflictos entre repositorios.
